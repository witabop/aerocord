# Aerocord website

Simple 2000s-style site for Aerocord. All images live in `docs/assets/` (flat list).

## Assets (copied from src/assets)

Copy these into `docs/assets/` so paths stay `assets/filename`:

- `TopBarBg.png` — from `src/assets/images/`
- `MainWnd.ico` — from `src/assets/images/icons/`
- `friend.ico` — from `src/assets/images/icons/`
- `wlm09banner.png` — from `src/assets/images/home/`
- `Logo.png` — from `src/assets/images/notification/` (optional)
- `defaultprofilepic.png` — from `src/assets/images/login/`
- `voicechannel.png` — from `src/assets/images/message/`
- `Active.ico` — from `src/assets/images/tray/`
- `LargeFrameOffline.png` — from `src/assets/images/frames/Frames/`

If you update any in `src/assets`, copy the file into `docs/assets/` again.

## GitHub Pages

**Settings → Pages → Source**: Deploy from branch **main**, folder **/docs**.

## Local preview

Open `index.html` in a browser or run a local server from `docs/`.
