Windows Live Messenger 2009 style Discord client.

What is Aerocord?

Aerocord is a custom Discord client for Windows that brings back the look and feel of Windows Live Messenger 2009. It is in essence a fork of Aerochat that rewrites the program as an electron app for greater portability across operating systems and ease of use. In its current state, Aerocord is more fully featured as a Discord client than Aerochat, but it still lacks some critical features that Aerochat had like the ability to run on older windows versions for instance.

Request a feature

Have an idea for Aerocord? Open an issue on GitHub to suggest it. Go to the Issues page, click New issue, and choose the feature-request template (or describe your idea). Or, if you are a developer, feel free to open a PR on the GitHub repository with your changes. We can’t promise every request will be implemented, but we do read them.
